Philosophy of a Blog

Writing brings up my questions about wether anything that I would write has any worth. Has someone else not said what I have and am going to say with better and more eloquent wording. Will any of it really matter? I say yes, because if I don't express myself in ways then I only prove that it doesn't matter. On the small, small chance that something that I say has some matter to someone else, then it does. Therefore I should share my opinion because if I don't then there is no point.

My Opinions

  • I think that everyone should keep an open mind.
  • I think that anyone should have and express their opinion without fear.
  • I think that technology has the potential to save us all, we need to choose how to use it.
  • I think that we need to make decisions to help ourselves and the world and that they are not conflicting goals.
  • I think that anyone can make a decision as long as they do not harm or take away rights from others.
  • I think that billionaires should not exist.
  • I think that we can save the world, but that is not possible alone.
  • I think that humans are naturally good.
  • I think that only though standardization can we create useful systems.
  • I think that (almost) all software should be open source.
  • I think that everyone should value people, as they are not means to an end.
  • I think that money should not exist, there are better ways.
  • I think that we need to focus on earth and not ourselves.
